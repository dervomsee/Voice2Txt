# Gemini CLI Agent Instructions

## Language Rules

*   **Code & Comments:** All code elements (variable names, functions, classes, etc.) and all inline/block comments MUST be written in **English**.
*   **User-Facing Strings:** All UI text, translatable strings, and messages visible to the end-user MUST be supported bilingually in **German (DE)** and **English (EN)** (e.g., using Android's `res/values/strings.xml` and `res/values-de/strings.xml`).
